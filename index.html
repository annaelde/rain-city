Project Synopsis

Scope

This website is my final project for the Web Scripting and AJAX course I took during the Summer 2017 semester.

The project requirements stated that students had to make a website about a city, its history, and its culture. The website would implement features that demonstrated an understanding of the class learning objectives, including: interacting with the DOM, using event handlers, interacting with APIs, using regular expressions, and form handling.

Before adding this project to my portfolio, I went through and revised the layout using Bootstrap. The most significant change I made was adding a hero image on each page; otherwise, the content is mostly the same.

Features

Original JavaScript implementations of mine that you'll find in the project include:

  • AJAX weather forecast using OpenWeatherMap API
  • Client-side contact form validation
  • Slideshow including captions
  • Cookie reader/writer and custom welcome message
  • Date time manipulation and time-dependent greeting
  • iFrame external website viewer
  • Animated image descriptions that appear on roll-over
